Java运算符一览

1.算术运算符：+，-，*，/，%，++，–

2.位运算符：&，|，~，^，<<，>>，>>>
(<<,>>,>>>分别是左移，右移，无符号右移。左移空位用0填充；右移空位用符号位填充；无符号右移用0填充，因为用0填充，所以结果总是一个正数)

(位运算符操作的是二进制数，将操作数转换二进制数后再进行运算，负数在计算机中以二进制补码的形式存在。符号位不变，原码取反后加一得出负数的补码。正数其实也是这样表示，只不过正数的原反补码都相同，不用进行转换。)

3.赋值运算符：=
(与其他运算符结合：+=，-=，*=，/=，%=，&=，|=，^=，<<=，>>=，>>>=)

4.比较运算符：>，>=，<，<=，==，!=（运算结果为布尔型：true or false）

5.逻辑运算符：&&，&，||，|，！，^
(与，不短路与，或，不短路或，非，异或)
(&& 与 &，|| 与 | 的区别：当 && / || 左边的操作数为 false / true 时，不计算右边的操作数，直接返回 false / true。而 & / | 两边的操作数都会计算。这便是不短路的意思。)

6.三目运算符 ： ? :
语法格式：(expression) ? (result1) : (result2)。expression为true返回result1，为false返回result2。
(三目运算符作用与if-else语句类似，运用得当能减少代码量)。
